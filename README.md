## 使用方法

| solo page 参数 | 类型    | 介绍                              |
| -------------- | ------- | --------------------------------- |
| room           | string  | 房间名（默认 `"cgb"`）            |
| auto           | boolean | 是否完全自动播放                  |
| velocity       | float   | 音乐进行速度                      |
| midi           | string  | 音乐文件（默认 `"我爱你中国.mid"` |
